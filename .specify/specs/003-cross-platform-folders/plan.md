# 实现计划：跨平台数据文件夹支持

## 技术栈

### 后端
- Python pathlib 跨平台路径处理
- 配置文件（JSON/YAML）
- 安全路径验证

### 前端
- 无需重大修改（已支持路径输入）

## 架构设计

### 路径验证流程
```
用户输入路径 → 转换为绝对路径 → 安全检查 → 验证存在性 → 检查 parquet 文件 → 返回结果
```

### 安全策略
1. 必须是绝对路径
2. 不能包含 `..` 路径遍历
3. 不能访问系统敏感目录
4. 必须是存在的目录
5. 必须包含 parquet 文件
