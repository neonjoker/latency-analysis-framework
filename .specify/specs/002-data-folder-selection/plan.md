# 实现计划：数据文件夹选择

## 技术栈

### 后端
- FastAPI 路由和端点
- Python pathlib 路径处理
- 安全路径验证

### 前端
- HTML 输入框和下拉选择
- JavaScript 路径验证
- 实时更新 UI

## 架构设计

### API 设计
```
GET  /api/folders          - 获取可用文件夹列表
POST /api/folders/validate - 验证文件夹路径
GET  /api/folders/current  - 获取当前选择的文件夹
```

### 数据流
```
前端选择 → API 验证 → 更新数据加载器 → 刷新日期列表 → 更新 UI
```

## 风险评估

| 风险 | 概率 | 影响 | 缓解措施 |
|------|------|------|----------|
| 路径遍历攻击 | 低 | 高 | 严格路径验证，限制在/mnt/f/ |

## 安全考虑

1. 路径必须以 `/mnt/f/` 开头
2. 禁止 `..` 路径遍历
3. 只允许访问存在的目录
